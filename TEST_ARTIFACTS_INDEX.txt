SUPPORT & TROUBLESHOOTING
================================================================================

Problem: verify.py says CORS still broken
  → Check line 54 of .env is: ALLOWED_ORIGINS=http://localhost:3000
  → NOT: ALLOWED_ORIGINS=["http://localhost:3000"]

Problem: Upload endpoint not rejecting type mismatch
  → Check rfp.py line 110-115 has validation code
  → Check line 134: zip(files, type_list) NOT zip(..., strict=False)

Problem: Dependencies not installing
  → cd apps/api
  → pip install -r requirements.txt
  → Ensure Python 3.11+

Problem: Database file not created
  → Check DATABASE_URL in .env: sqlite+aiosqlite:///./entropy_dev.db
  → Run python seed.py to create it

Problem: Can't start API
  → Check port 8000 is not in use
  → Run: netstat -ano | findstr 8000
  → Kill any process: taskkill /PID <PID> /F
